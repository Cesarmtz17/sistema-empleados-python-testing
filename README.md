# Sistema de Empleados - Python + Testing

Sistema de gestión de empleados con **POO** y suite completa de tests usando pytest.

## Clases
- `Empleados` - Clase base
- `Gerente` - Aumentos de sueldo
- `Tecnico` - Horas extras
- `EmpleadoVentas` - Comisiones

## Ejecutar Tests
```bash
pip install pytest
pytest -v
```
